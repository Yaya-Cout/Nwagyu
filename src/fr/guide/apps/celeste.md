# Celeste Classic

Connaissez-vous Celeste, le jeu de plateforme? Un fait peu connu à propos de ce
jeu est qu'il a été à l'origine développé en 4 jours pour la console virtuelle
PICO-8, avant d'être renommé Celeste Classic.

Et si vous pouviez jouer à Celeste Classic sur votre calculatrice,
gratuitement ? Grâce à l'adaptation de [BenchatonDev](https://github.com/BenchatonDev),
c'est désormais possible !

## Comment jouer

Les contrôles peuvent sembler complexe à première vue, mais une fois que vous
les comprenez, ils deviennent plutôt simples à retenir :

| Action                                       | NumWorks       |
| -------------------------------------------- | -------------- |
| Dash                                         | OK             |
| Sauer                                        | Retour         |
| Se déplacer                                  | Flèches        |
| Pause                                        | Backspace      |
| Sauvegarder                                  | Shift          |
| Charger la sauvegarde                        | Alpha          |
| Désactiver les tremblements d'écrans         | Boite à outils |
| Réinitialiser                                | XNT            |
| Quitter                                      | Home           |
